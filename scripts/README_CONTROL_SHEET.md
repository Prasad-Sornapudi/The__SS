# Control Sheet Secrets Obfuscation

This directory contains scripts for obfuscating the control sheet credentials for secure embedding in the APK.

## New Implementation (Single Control Sheet)

The new implementation uses a single **App_Control Google Sheet** as the source of truth instead of embedding individual sheet links.

### Requirements

1. **App_Control Google Sheet** with the following structure:
   - `Login_Credentials` tab with columns: `User_Name`, `Password`
   - Individual batch tabs (one per batch):
     * `Skill_Sync01` tab with configuration columns:
       - `Master_Sheet_Link`
       - `Master_Sheet_Credentials`
       - `Attendance_Sheet_Link`
       - `Attendance_Sheet_Credentials`
       - `Mock_Interview_Sheet_Link`
       - `Mock_Interview_Sheet_Credentials`
       - `Department_Sheet_Link`
       - `Department_Sheet_Credentials`

2. **Control Service Account JSON** - A service account with read access to the control sheet

### Usage

```bash
# Generate obfuscated chunks for control sheet
python obfuscate_control_sheet_secrets.py \
  --control-sheet-url "https://docs.google.com/spreadsheets/d/YOUR_CONTROL_SHEET_ID/edit" \
  --control-service-account "path/to/control_service_account.json" \
  --chunks 3 \
  --output-dir "../assets/secrets"
```

### Output

The script generates:
- `secret_part_1.txt`, `secret_part_2.txt`, `secret_part_3.txt` (obfuscated chunks)
- `secret_hmac.txt` (integrity verification token)

These files are embedded in the APK resources.

### Security Notes

1. Only embed obfuscated chunks generated by this script at release time
2. Never check raw JSON or control URL into source control
3. Use ProGuard/R8 obfuscation
4. Use non-obvious resource keys
5. Disable reconstruction in debuggable builds
6. Minimize permissions of the embedded service account
7. Rotate Control SA keys and regenerate chunks periodically

### Migration from Previous Implementation

To migrate from the previous per-class embedding approach:

1. Remove all existing per-class embedded chunks from assets/secrets
2. Create an App_Control Google Sheet with the new batch-tab-driven structure
3. Create individual tabs for each batch with the required configuration columns
4. Run this script with the control sheet URL and service account
5. Rebuild the APK
